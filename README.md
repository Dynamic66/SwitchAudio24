How to Use:

   Outputs can be switched by pressing s + alt or double-clicking the tray icon.   ![image](https://github.com/Dynamic66/SwitchAudio24/assets/160179474/0b2f9ab0-8586-42e6-b999-1aba361318a7)

----------------------------------------------------
Setup:

   1. Download the zip file and extract the folder to the desired location.

   2. Start Switchaudio24.exe as an admin.
   
On the first startup, the Settings will open first.

1. Choose the order of outputs and Check the outputs that you want to switch beween.
 
   ![image](https://github.com/Dynamic66/SwitchAudio24/assets/160179474/952f29ce-9516-470d-80e6-9f01cbbd9296)

2. Click in the "Hotkey" Text box and press a desired key combination (minimum 1 key, maximum 1key and 1 modifier).

   ![image](https://github.com/Dynamic66/SwitchAudio24/assets/160179474/6c885685-2475-4d38-8b50-171ed2c8098d)
   
3. Check the "Open Switchaudio24 at startup" if you want the Application to run after starting your Computer.
 
   ![image](https://github.com/Dynamic66/SwitchAudio24/assets/160179474/04dad682-a8c8-486d-873d-181dc274542b)


   
Settings can be accessed by Right-click the tray icon and selecting "Settings".

   ![image](https://github.com/Dynamic66/SwitchAudio24/assets/160179474/5f119e57-307b-466d-9a64-1b485ef01c52)
